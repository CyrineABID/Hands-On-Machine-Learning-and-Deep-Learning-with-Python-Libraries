# Hands-On Machine Learning and Deep Learning with Python Libraries <br>
This repository contains Jupyter notebooks designed to help beginners and intermediate learners explore machine learning (ML) and deep learning (DL) using popular Python libraries. It provides step‑by‑step examples, explanations, and visualizations to make concepts easy to understand.

# What’s Inside

Introduction to essential Python libraries:
* NumPy – numerical computing
* Pandas – data manipulation
* Matplotlib – data visualization
* Seaborn – statistical plots
* Scikit-learn – classical ML algorithms
* TensorFlow/Keras – deep learning

# 🛠️ Requirements <br>
* Python 3.8+
* Jupyter Notebook or JupyterLab
* The code was written in Python language (v.4.5.1) using the Anaconda distribution (v.2.7 ; 12-2-2025) and Jupyter Notebook tool.

## 📂 Datasets

This project uses a mix of **built‑in datasets** (from `scikit-learn` and `seaborn`) and **popular benchmark datasets** for classification, regression and deep learning tasks. These are chosen because they are small, easy to understand, and widely used in tutorials.

### Commonly used datasets
- **Iris dataset** – flower classification (from `sklearn.datasets`)
- **Digits dataset (MNIST)** – handwritten digit recognition (from `sklearn.datasets` or `keras.datasets`)

## Learning Path

This notebook is the second step in the series:

1. *Basics* → Python structures and ML workflow
2. *Libraries* → NumPy, Pandas, Matplotlib, Scikit-learn, TensorFlow, etc.
3. *Regression* → KNN with a major emphasis on the philosophy of explanatory analysis and preprocessing
4. *Classification* → Logistic regression, decision trees, SVM, XGboost.
5. *Deep Learning Basics* → Neural network architectures and training
