# Basics: Hands-On Machine Learning and Deep Learning with Python Libraries

## 📘 Overview <br>
This notebook introduces the **fundamentals of Python programming** and their application in machine learning (ML) and deep learning (DL). It serves as the starting point in a series of notebooks that progressively cover libraries, regression, classification, and deep learning concepts.

The focus here is on building a strong foundation with Python data structures and understanding how they connect to ML/DL workflows.

## 📂 Contents <br>
- **Lists** → Ordered, mutable collections for handling datasets.
- **Dictionaries** → Key–value mappings for configurations, label encoding, and structured data.
- **Tuples** → Immutable sequences often used for fixed data or function outputs.
- **Basic ML Workflow** → Introduction to data preprocessing, model training, and evaluation.
- **Transition to Libraries** → How lists, dictionaries, and tuples lead into NumPy arrays and Pandas DataFrames.

## ⚙️ Requirements <br>
The code was written in Python language (v.4.5.1) using the Anaconda distribution (v.2.7 ; 12-2-2025) and Jupyter Notebook tool.
