# GTCA_BEST_Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries
As part of the GTCA-BEST educational program, I had the pleasure of being invited as a keynote speaker at the second "Junior Talent Session" to lead a **30 minutes** hands-on workshop on **Python libraries** applied to machine learning and deep learning algorithms. <br>

Code execution was carried out in Python 3 using Jupyter Notebook, with dependencies managed via the Anaconda distribution [v.2.7 (12-2-2025)] on a local machine

# Overview <br>
This repository contains a series of Jupyter notebooks designed to introduce and practice machine learning (ML) and deep learning (DL) concepts using Python’s most popular libraries. The notebooks provide a hands-on approach, guiding learners from foundational topics to advanced techniques.

# Notebooks Included <br>
* **Basics-Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries.ipynb** <br>
      Introduction to Python data structures (lists, dictionaries, tuples) and ML/DL workflow basics.
* **Libraries-Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries.ipynb** <br>
      Overview of essential Python libraries such as NumPy, Pandas, Matplotlib, Scikit-learn, TensorFlow, etc.
* **Regression-Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries.ipynb** <br>
      Covers regression techniques (linear regression, polynomial regression, evaluation metrics).
* **Classification-Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries.ipynb** <br>
      Explains classification algorithms (logistic regression, decision trees, random forests, neural networks).
* **DeepLearning-Basics-Hands-On-Machine-Learning-and-Deep-Learning-with-Python-Libraries.ipynb** <br>
      Provides an introduction to deep learning fundamentals, including neural network basics, activation functions, forward and backward propagation, and simple model training examples using TensorFlow.

# Learning Path

1. *Start with Basics* → Learn Python structures and ML workflow.
2. *Explore Libraries* → Get familiar with the tools powering ML/DL.
3. *Dive into Regression & Classification* → Practice supervised learning.
4. *Advance to Deep Learning* → Build and train neural networks.

# Audience <br>
* Beginners in machine learning and deep learning.
* Students seeking practical, hands-on experience.
* Developers transitioning into AI/ML fields.
